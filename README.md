
# Ballot - DAPP Tutorial
This migrates all of the frontend code of this tutorial to use ReactJS, Es6, & Webpack

## Getting Started
```
$ npm install
$ npm install -g webpack webpack-cli
$ truffle init
$ truffle migrate
$ webpack src/js/App.js -o ./bundle.js
$ npm run start
```
